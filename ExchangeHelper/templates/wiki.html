Администратор
Интерфейс администратора состоит из следующих пунктов (более подробная информация расписана в конце):
  • 1. Рабочее место (для перехода нажмите на изображение слева сверху)
    • На главной странице располагаются следующие элементы:
    • 1. "Создать новую кассу";
    • 2. Cписок всех доступных для работы касс.
  • 2. Личный кабинет
    • В личном кабинете администратора располагаются следующие элементы:
    • 1. "Создать новую кассу";
    • 2. "Записки";
    • 2. Cписок всех записок, в хронологическом порядке, и интерфейс для их удаления.
  • 3. Список всех касс
    • В выпадающем списке всех касс администратор может выбрать интересующую его кассу и перейти к более детальному просмотру;
  • 4. Главная страница
    • Краткое описание проекта;
  • 5. Помощь
    • Страница с помощью юзеру;
  • 6. Отчёт
    • Страница с отчётами о работе касс их операциями и прибылями;
  • 7. Выход
    • Кнопка Выхода/Входа в аккаунт;
Подробное описание всех элементов и возможностей администратора
  • 1. Касса
    • В кассе администратора, так же как и в любой другой кассе для пользователя со специальными правами, предоставлены следующие возможности в меню:
    • 1. "Подкрепление"
      • Администратор имеет право выделить определённую сумму для работы кассы. Для этого он должен нажать на кнопку "Подкрепление", в появившемся окне обязательно ввести сумму подкрепления, выбрать валюту и при желании написать коментарий к данной операции, затем нажать кнопку "Поддержать" и введённая сумма окажется на балансе открытой кассы. Для отмены следует нажать кнопку "Закрыть";
    • 2. "Инкассация"
      • Инкассация - обратное действие подкрепления, администратор имеет право инкассировать определённую сумму из кассы. Для этого он должен нажать на кнопку "Инкассация", в появившемся окне обязательно ввести сумму инкассации, выбрать валюту и при желании написать коментарий к данной операции, затем нажать кнопку "Утвердить" и введённая сумма спишется со счёта открытой кассы. Для отмены следует нажать кнопку "Закрыть";
    • 3. "Траты на кассу"
      • Траты на кассу - администратор имеет возможность выделить определённую сумму из главной кассы на определённые (бытовые и иные) нужды другой кассы. Для этого он должен нажать на кнопку "Траты на кассу", в появившемся окне обязательно ввести сумму выделений, выбрать валюту и обязательно написать коментарий к операции, затем нажать кнопку "Утвердить" и введённая сумма спишется со счёта главной кассы. Для отмены следует нажать кнопку "Закрыть";
    • 4. "Записки"
      • Интерфейс который предостваляется не только лишь администратору, но и обычному пользователю и является интерфейсом односторонней связи "рядовой кассир -> администратор". Для создания новой записки требуется нажать на "Записки" и в появившемся окне заполнить причину записки, при надобности также введите требуемую сумму, валюту и обязательно напишите коментарий, затем нажмите кнопку "Отправить". Для отмены следует нажать кнопку "Закрыть". Если записка была создана то у администратора рядом с надписью "Личный кабинет" появится оповещение в виде цифры - колличества активных записок, при переходе в кабинет - отобразится список всех записок и появится возможность для их удаления.
    • 5. Курс валют
      • Под меню юзера находится таблица с курсом валют, а у администратора ещё и интерфейс для изменения курса. Для изменения курса потребуется нажать кнопку "Изменить" и в появившемся окне изменить прежний курс валют на акутаульный, затем нажать кнопку "Сохранить изменения". Для отмены следует нажать кнопку "Закрыть". При изменении курса валют чистая прибыль кассы будет пересчитана для актуального курса.
    • 6. Редактирование
      • Под названием кассы расположена кнопка для редактирования аккаунта кассы (подробнее про это написано ниже - "Редактирование кассы").
    • 7. Доступные средства
      • Под названием кассы расположена таблица с актуальным остатком денег в кассе.
    • 8. Форма создания новой операции
  • 2. "Создать новую кассу"
    • При нажатии на кнопку для создания новой кассы, администратор переходит на страницу с частично заполненными формами. Для корректного создания кассы администратор должен обязательно заполнить поля:
    • Логин - вписывается данные для входа пользователя;
    • Пароль - пароль для входа пользователя;
    • Полное описание кассы - полное описание кассы, рекомандуется вписывать общую информацию о кассе, её место положение и другую важную информацию;
    • Краткое название - краткое название кассы, по которому будет легко её идентифицировать в списке с остальными элементами;
    • Таблица с курсами валют не обязательна к заполнению - присутствует таблица со списком валют, ценой покупки, ценой продажи основных валют в кассе, может быть заполнена актуальным курсом валют.
    После окончания заполнения всех полей
