Администратор
Интерфейс администратора состоит из следующих пунктов (более подробная информация расписана после списка элементов):
  • 1. Рабочее место
    • На главной странице располагаются следующие элементы:
    • 1. "Создать новую кассу";
    • 2. Cписок всех доступных для работы касс.
  • 2. Личный кабинет
    • В личном кабинете администратора располагаются следующие элементы:
    • 1. "Создать новую кассу";
    • 2. "Записки";
    • 2. Cписок всех записок, в хронологическом порядке, и интерфейс для их удаления.
  • 3. Касса администратора
    • В кассе администратора, так же как и в любой другой кассе для пользователя со специальными правами, предоставлены следующие возможности в меню пользователя:
    • 1. "Подкрепление";
    • 2. "Инкассация";
    • 3. "Записки";
    • 4. В таблице с курсом валют присутствует возможность изменить курс;
    • 5. Под кратким описанием-названием кассы присутствует кнопка "Редактировать", которая при нажатии направляет администратора в интерфейс для редактирования информации о кассе;
    • 6. В такблице "Произведённые операции" в последнем столбце присутствует интерфейс для удаления выполненных операций.
Подробное описание всех элементов и возможностей администратора
  • "Создать новую кассу". При нажатии на кнопку для создания новой кассы, администратор переходит на страницу с частично заполненными формами. Для корректного создания кассы администратор должен обязательно заполнить поля:
    • Логин - вписывается данные для входа пользователя;
    • Пароль - пароль для входа пользователя;
    • Полное описание кассы - полное описание кассы, рекомандуется вписывать общую информацию о кассе, её место положение и другую важную информацию;
    • Краткое название - краткое название кассы, по которому будет легко её идентифицировать в списке с остальными элементами;
    • Таблица с курсами валют не обязательна к заполнению - присутствует таблица со списком валют, ценой покупки, ценой продажи основных валют в кассе, может быть заполнена актуальным курсом валют.
    После окончания заполнения всех полей
